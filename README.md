# shopping-cart
這是一個透過Vue CLI所建立的網拍專案，主要功能是購物車的使用
（素材來源：Pazzo）

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

## Install Vue CLI
可以參考這個網頁的教學：
```
https://book.vue.tw/CH3/3-1-vue-cli-introduction.html
```
